HUA-010

Gestión de carreras y universidades

Usuario Administrador
Prioridad Alta
Riesgo Medio
Responsable Tobar Sebastián

Descripción

Como administrador, necesito gestionar la base de datos de carreras y universidades públicas y privadas, para mantener actualizada la oferta académica disponible en el sistema.

Observación

La gestión incluye instituciones de tercer nivel, institutos tecnológicos e institutos superiores universitarios, tanto públicos como privados.

Criterios de aceptación

  • El sistema debe permitir registrar, editar y eliminar carreras asociadas a una o varias instituciones.
  • El sistema debe validar que no se dupliquen registros de una misma carrera dentro de la misma institución.
  • El sistema debe reflejar los cambios realizados de forma inmediata en las recomendaciones mostradas a los postulantes.
