HUA-007

Consultar puntajes referenciales del Senecyt

Usuario Postulante
Prioridad Media
Riesgo Medio
Responsable Sebastian Tobar

Descripción

Como postulante, necesito consultar los puntajes referenciales de postulación de ciclos anteriores, para conocer los requisitos aproximados de las carreras que me interesan.

Observación

La información proviene de una base de datos cargada previamente a partir de fuentes oficiales del Senecyt y no se actualiza en tiempo real.

Criterios de aceptación

  • El sistema debe permitir filtrar los puntajes referenciales por carrera y por institución.
  • El sistema debe indicar el periodo académico al que corresponde cada puntaje mostrado.
  • El sistema debe mostrar un mensaje claro cuando no existan datos disponibles para una búsqueda.
