HUA-003

Recuperar contraseña

Usuario Postulante
Prioridad Media
Riesgo Bajo
Responsable Tobar Sebastián

Descripción

Como postulante, necesito recuperar mi contraseña mediante mi correo electrónico registrado, para no perder el acceso a mi cuenta en caso de olvido.

Observación

El enlace de recuperación debe tener un tiempo de expiración limitado por motivos de seguridad.

Criterios de aceptación

  • El sistema debe enviar un enlace de recuperación al correo registrado del usuario.
  • El enlace debe expirar luego de 30 minutos de haber sido generado.
  • El sistema debe permitir establecer una nueva contraseña que cumpla las reglas de seguridad.
