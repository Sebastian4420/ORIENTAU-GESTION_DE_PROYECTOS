HUA-004

Realizar test de aptitudes

Usuario Postulante
Prioridad Alta
Riesgo Alto
Responsable Tobar Sebastián

Descripción

Como postulante, necesito realizar un test dividido en cinco áreas de conocimiento (verbal, numérica, lógica, memoria y atención), para obtener una medición objetiva de mis fortalezas y aptitudes.

Observación

El test debe poder guardarse por sección, en caso de que el postulante no complete todas las áreas en una sola sesión.

Criterios de aceptación

  • El sistema debe presentar preguntas agrupadas por cada una de las cinco áreas de evaluación.
  • El sistema debe calcular automáticamente un puntaje por área al finalizar el test.
  • El sistema debe impedir el reenvío de un test ya finalizado sin autorización del administrador.
