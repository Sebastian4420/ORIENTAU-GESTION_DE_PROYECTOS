HUA-006

Visualizar informe de afinidad académica

Usuario Postulante
Prioridad Alta
Riesgo Alto
Responsable Jostin Morales

Descripción

Como postulante, necesito visualizar un informe con las carreras y universidades recomendadas según mis resultados, para tener una idea clara sobre qué estudiar.

Observación

El informe debe presentar varias opciones de carrera, evitando limitar la recomendación a una única alternativa.

Criterios de aceptación

  • El sistema debe generar el informe automáticamente al finalizar el test y el cálculo de puntaje.
  • El informe debe mostrar al menos tres opciones de carrera ordenadas por nivel de afinidad.
  • El sistema debe permitir descargar o visualizar nuevamente el informe generado.
