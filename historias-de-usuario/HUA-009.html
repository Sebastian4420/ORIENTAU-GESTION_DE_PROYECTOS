HUA-009

Gestión de preguntas del test

Usuario Administrador
Prioridad Alta
Riesgo Medio
Responsable Sebastian Tobar

Descripción

Como administrador, necesito crear, editar, consultar y eliminar las preguntas del test de aptitudes, para mantener actualizado y depurado el banco de evaluaciones.

Observación

Cada pregunta debe estar asociada a una de las cinco áreas de conocimiento evaluadas.

Criterios de aceptación

  • El sistema debe permitir asociar cada pregunta a un área específica de evaluación.
  • El sistema debe impedir eliminar preguntas que ya hayan sido respondidas por algún postulante, permitiendo únicamente desactivarlas.
  • El sistema debe validar que los campos obligatorios de la pregunta estén completos antes de guardarla.
