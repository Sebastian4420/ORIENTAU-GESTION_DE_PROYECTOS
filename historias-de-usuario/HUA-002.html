HUA-002

Iniciar y cerrar sesión

Usuario Postulante
Prioridad Alta
Riesgo Medio
Responsable Morales Jostin

Descripción

Como postulante, necesito iniciar y cerrar sesión de forma segura, para garantizar el acceso controlado a mis evaluaciones y resultados personales.

Observación

El acceso se realiza mediante correo y contraseña previamente registrados; no se contempla inicio de sesión mediante redes sociales.

Criterios de aceptación

  • El sistema debe validar las credenciales antes de conceder acceso.
  • El sistema debe bloquear el acceso luego de tres intentos fallidos consecutivos, solicitando esperar diez minutos.
  • El cierre de sesión debe invalidar el token de acceso activo.
