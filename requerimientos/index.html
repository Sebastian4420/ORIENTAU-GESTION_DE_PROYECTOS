Postulante

ID Descripción
RR-001 Como postulante, quiero registrarme ingresando nombre, correo y contraseña, para crear mi perfil de usuario en el sistema.
RR-002 Como postulante, quiero iniciar y cerrar sesión de forma segura, para acceder de manera controlada a mis evaluaciones y resultados.
RR-003 Como postulante, quiero recuperar mi contraseña mediante correo electrónico, para no perder el acceso a mi cuenta.
RR-004 Como postulante, quiero realizar un test de aptitudes en las cinco áreas de conocimiento (verbal, numérica, lógica, memoria, atención), para descubrir mis fortalezas y habilidades.
RR-005 Como postulante, quiero ingresar mi promedio de grado final, para obtener una estimación referencial de mi puntaje de postulación.
RR-006 Como postulante, quiero visualizar un informe de afinidad académica con carreras y universidades recomendadas, para tomar una decisión más informada sobre mi futuro académico.
RR-007 Como postulante, quiero consultar la base de puntajes referenciales de ciclos anteriores obtenida del Senecyt, para conocer los requisitos aproximados de las carreras de mi interés.

Orientador / Docente

ID Descripción
RR-008 Como orientador, quiero acceder a los resultados de las evaluaciones de los estudiantes que asesoro, para brindarles una orientación personalizada en la elección de carrera.

Administrador

ID Descripción
RR-009 Como administrador, quiero crear, editar, consultar y eliminar las preguntas del test de aptitudes, para mantener actualizado y depurado el banco de evaluaciones.
RR-010 Como administrador, quiero gestionar la base de datos de carreras y universidades públicas y privadas, para mantener actualizada la oferta académica disponible en el sistema.
